# AMIC Merged Project

هذا المشروع المدمج يحتوي على:

- `modern-app/`: النسخة الحديثة (React + Vite + Supabase)
- `legacy-forms/`: النسخة الكلاسيكية (HTML/JS) للاحتفاظ بكل الميزات القديمة

## التشغيل

### modern-app

1. افتح مجلد `modern-app`
2. نفذ تثبيت الحزم
3. شغل التطبيق

### legacy-forms

ملفات ثابتة HTML/JS ويمكن تشغيلها عبر أي استضافة static أو فتح الصفحات مباشرة.

## ملاحظة الدمج

تم الدمج على شكل Monorepo بسيط بدون حذف أي جزء من أي مشروع.
النسخة الحديثة هي الأساس للتطوير القادم، والنسخة الكلاسيكية محفوظة للمقارنة أو النقل التدريجي.
